Sistema web desarrollado a medida para la gestión integral de bienes inmuebles, arrendamientos, generación de documentos financieros (comprobantes de egreso y facturas), y control administrativo. La solución está diseñada específicamente para las necesidades de Agustin Lozano Inmobiliaria, permitiendo administrar de manera eficiente y profesional su portafolio de 4 locales comerciales con posibilidad de escalar a futuro
